# MouseView
Support remote control mouse or TV remote control into the mouse(支持遥控器操作的的鼠标和遥控器做鼠标来用)。

**实现功能**： 遥控器方向键控制鼠标箭头移动
             遥控器可以操作模拟鼠标进行点击操作。
